## Features
* Slider für Beiträge
* Individuelle Slider Einstellungen
* Galerie für Beiträge
* Individuelle Galerie Einstellungen
* Beiträge als News anzeigen

== Description ==
Auswahl von Beiträgen im Gutenberg-Block Editor mit verschiedenen Ausgabeoptionen.
